Bem-vindo ao meu repositório dedicado à disciplina de **Linguagem de Programação II**, parte da grade do curso de Análise e Desenvolvimento de Sistemas da **FATEC Sorocaba**.

Neste repositório, documento minha jornada de aprendizado e evolução na linguagem **C#** e no ecossistema **.NET**. O foco principal desta matéria é expandir a base da lógica de programação para o desenvolvimento de soluções mais robustas, unindo os pilares da Orientação a Objetos com a criação de aplicações funcionais.

---

## 🎯 Objetivos

Os códigos e projetos deste repositório refletem uma progressão de complexidade, indo desde conceitos estruturais de back-end até a construção de aplicações com interface gráfica. As principais áreas exploradas incluem:

- **Aprofundamento em POO com C#:** Aplicação prática de encapsulamento, herança, polimorfismo, classes abstratas e interfaces.
- **Manipulação de Dados:** Uso de coleções, listas genéricas e estruturas de controle para gerenciamento eficiente de informações em memória.
- **Desenvolvimento de Interfaces (GUI):** Construção de aplicações desktop utilizando **Windows Forms**, conectando a lógica de negócio estruturada no back-end com a experiência do usuário final.
- **Integração com Banco de Dados:** Persistência de dados conectando a aplicação C# ao banco de dados **MySQL**.
- **Boas Práticas:** Foco em modularidade, separação de responsabilidades (camadas) e escrita de código limpo e reaproveitável.

## 📁 Estrutura do Repositório

- `\Atividades`: Exercícios e desafios práticos para fixação da sintaxe e lógica em C#.
- `\Projetos`: Aplicações desktop completas com interface gráfica (Windows Forms) e integração com banco de dados.
- `\Provas`: Resoluções das avaliações práticas da disciplina.

## 🛠️ Pré-requisitos

Para compilar e executar as soluções deste repositório, você precisará instalado em sua máquina:

- **.NET SDK**: Versão correspondente aos projetos (geralmente instalada junto ao Visual Studio).
- **Visual Studio**: Recomendado (Community ou superior) com a carga de trabalho de *Desenvolvimento para desktop com .NET* instalada.
- **Banco de Dados**: MySQL Server e MySQL Workbench (para os projetos com persistência de dados).

## 🚀 Como Executar

1. Clone este repositório em sua máquina local:
```bash
   git clone [https://github.com/Romerac/Linguagem-de-Programacao-2-CSharp.git](https://github.com/Romerac/Linguagem-de-Programacao-2-CSharp.git)
```

2. Abra o arquivo de solução (.sln) ou a pasta do projeto no Visual Studio.

3. Caso o projeto utilize banco de dados, certifique-se de executar o script .sql correspondente no seu servidor MySQL local e validar a string de conexão (Connection String) no código C#.

4. Pressione F5 ou clique no botão Iniciar/Start no Visual Studio para compilar e rodar a aplicação.
